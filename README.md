# 従業員管理アプリ（Employee Management App）

Java（Swing）と SQLite を使用した、シンプルなデスクトップ従業員管理アプリです。
未経験から Java エンジニア／テストエンジニアを目指す方向けに、**基本構成・設計分離・DB操作**を意識して実装しています。

---

## 📌 概要

本アプリは、従業員情報を **登録・一覧表示・削除** できるデスクトップアプリケーションです。
GUI には Swing、データ永続化には SQLite（JDBC）を使用しています。

---

## 🛠 使用技術

* Java 17（※ Java 8 以上でも可）
* Swing（GUI）
* SQLite
* JDBC
* Eclipse

---

## 📂 構成

```
src/
 ├─ Main.java          // 画面（GUI）クラス
 ├─ Employee.java      // 従業員モデルクラス
 ├─ EmployeeDAO.java   // DB操作クラス（DAO）
 └─ DBUtil.java        // DB接続・初期化ユーティリティ
```

---

## 🧱 設計ポイント

* **画面（Main）とDB処理（DAO）を分離**
* SQLite を使用し、アプリ起動時に自動でテーブルを生成
* `PreparedStatement` を使用し SQL インジェクション対策
* `try-with-resources` によるリソース管理
* `LocalDate` を使用した日付管理

---

## ✨ 機能一覧

### 1️⃣ 従業員登録

* ID / 氏名 / 部署 / 入社日 を入力して登録
* 登録後、入力欄は自動でクリア
* SQLite データベースに保存

### 2️⃣ 一覧表示

* DB に登録されている従業員情報を一覧表示
* JTable によるテーブル表示
* 列クリックで昇順・降順ソート可能

### 3️⃣ 削除

* 一覧から行を選択して削除
* 削除前に確認ダイアログを表示

## 🚀 実行方法

1. SQLite JDBC ドライバをプロジェクトに追加
2. Eclipse でプロジェクトをインポート
3. `Main.java` を実行

DB ファイルは以下の場所に自動作成されます。

```
C:/data/employee.db
```
